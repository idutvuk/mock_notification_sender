# Photo Point
# Техническое задание: Система уведомлений

## Задача

Создать сервис для отправки уведомлений пользователям.

## Требования

Пользователь может получать уведомления через:

- Email
    
- SMS
    
- Telegram
    

Нужно обеспечить надежную доставку - если один способ не сработал, попробовать другой.

## Запусе

1. Скачайте `uv`: [по ссылке](astral.sh.com)
2. Запустите `uv sync`
3. `uv run main.py` - по ссылке будет доступна документация, в которой можно дёрнуть эндпойнты для задачи

## Использование

1. Эндпойнт `notify` при пост запросе выдаёт id таски, которая запускается при выполнении

2. Можно обратиться к серверу (get `/jobs/<id>` ) чтобы узнать статус задачи
